**Create reasoning/app/nodes/valuation.py:**

1. Define `valuation_node(state: ReportState) -> dict` function
2. Branch on `state["asset_type"]`:

**Equity path:**
- Read `state["fundamentals_rows"]`, `state["regime_analogues"]`, `state["earnings_docs"]`
- Compute regime-relative metrics:
  - For P/E: take top 2-3 analogues sorted by similarity_score, compute weighted average analogue P/E (if available from analogue narrative or metadata), compare current P/E vs weighted analogue average
  - For P/B: same approach
  - If P/E or P/B is None on current stock, add to missing_metrics, compute what's available
- Build prompt with formatted fundamentals + analogue context using prompts.py helpers
- Call Gemini `with_structured_output(ValuationOutput)` for narrative generation
- Assign valuation_label: Attractive (current significantly below analogue avg), Fair (within range), Stretched (significantly above)
- Populate sources: map pe_ratio -> "fundamentals:{symbol}:{data_as_of}", pe_vs_analogue_avg -> "computed:analogue_weighted_avg:{analogue_ids}"

**Gold path:**
- Read `state["fred_rows"]` (extract real yield: FEDFUNDS - CPIAUCSL or similar proxy), `state["gold_etf_rows"]`, `state["gold_price_rows"]`
- Read `state["macro_regime_output"]` if present (for macro overlay context — per locked decision: gold valuation benefits from macro regime context)
- Compute real_yield from FRED data
- Summarize ETF flow context from GoldEtfRow volume trends
- Always add WGC warning: "WGC central bank buying data unavailable (HTTP 501) — gold valuation excludes central bank demand context"
- Call Gemini `with_structured_output(ValuationOutput)` for narrative
- Assign valuation_label based on real yield environment and price momentum

**Both paths:**
- Populate sources dict for all numeric fields
- Propagate warnings from input data types
- Return `{"valuation_output": result}`

**CRITICAL constraints per CONTEXT.md locked decisions:**
- Gold and equity are SEPARATE assessment paths (not unified)
- When fundamental data is missing, produce PARTIAL assessment — do NOT skip entirely
- WGC data is flagged as DATA WARNING per existing convention

- Mock Gemini LLM calls using `unittest.mock.patch`
- Test all 9 behaviors listed above
- For equity analogue weighting test: provide 2 analogues with different similarity_scores and verify the weighted average is not a simple mean
