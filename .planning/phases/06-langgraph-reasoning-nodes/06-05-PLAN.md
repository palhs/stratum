**Create reasoning/app/nodes/grounding_check.py:**

1. Define helper `_collect_float_fields(model: BaseModel, prefix: str) -> list[tuple[str, str]]`:
   - Walk model.model_fields and check values
   - For each field where `isinstance(getattr(model, field_name), float)` (not None): yield `(prefix.field_name, field_name)`
   - For nested Pydantic models: recurse with extended prefix
   - For list[Pydantic] fields (like regime_probabilities): iterate and recurse with index prefix
   - Skip fields that are None, str, bool, int, list[str], dict

2. Define helper `_verify_output(output: BaseModel, output_key: str, errors: list[str]) -> None`:
   - Get `sources = getattr(output, "sources", {})`
   - For each float field from `_collect_float_fields`:
     - Check if the short field name (last component) exists in sources
     - For nested models in lists (e.g., RegimeProbability): check if the model has its own source field (source_analogue_id) as alternative to sources dict
   - If not found: append error message to errors list

3. Define `grounding_check_node(state: ReportState) -> dict`:
   - Define which state keys to check: `["macro_regime_output", "valuation_output", "structure_output"]`
   - For each key: if state.get(key) is not None, call _verify_output
   - Track checked_outputs list
   - If errors exist: raise `GroundingError(f"Grounding check failed — {len(errors)} unattributed numeric claims:\n" + "\n".join(errors))`
   - If no errors: return `{"grounding_result": GroundingResult(status="pass", checked_outputs=checked_outputs)}`

**CRITICAL constraints per CONTEXT.md locked decisions:**
- Verify BOTH raw database values AND derived calculation chains
- Qualitative claims need data SOURCE attribution (e.g., "based on FRED indicators") — NOT record ID. This means str fields are not checked for record IDs.
- Fail mode: raise GroundingError (explicit error per SC #5), not a warning
- Only float fields and derived calculations require record ID citations
- Do NOT regex-scan narrative strings — check only typed Pydantic fields
