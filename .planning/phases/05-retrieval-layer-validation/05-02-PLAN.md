    **2. LLM-parameterized retrieval via CypherTemplateRetriever (for Phase 6 integration):**
    ```python
    def get_regime_analogues(query_text: str, limit: int = 5, graph_store=None, llm=None) -> list[RegimeAnalogue]:
        """Retrieve regime analogues matching a natural language macro query."""
    ```
    Uses CypherTemplateRetriever with a Pydantic RegimeParams model. The LLM fills `regime_keywords` from the query text. Cypher template searches Regime nodes by keyword match on name/notes, traverses HAS_ANALOGUE, returns full relationship properties.

    **Cypher templates to implement:**
    1. `REGIME_ANALOGUE_BY_KEYWORD` — Matches regimes by keyword in name/notes fields, traverses HAS_ANALOGUE, returns analogue with all relationship properties
    2. `ALL_ANALOGUES` — Returns all HAS_ANALOGUE relationships (no params needed)
    3. `REGIME_BY_FRED_RANGE` — Matches regimes where FRED indicator values fall within a range (for fallback when keyword search fails, per Pitfall 2 mitigation)

    - Accept optional `graph_store` parameter; if None, create from env vars (NEO4J_URI, NEO4J_PASSWORD)
    - Use `Neo4jPropertyGraphStore(username="neo4j", password=..., url=...)` — NOT PropertyGraphIndex (Pitfall 6)
    - Validate graph connectivity on first call: `structured_query("MATCH (n:Regime) RETURN count(n)")` must return > 0

    **Logging:** Use Python `logging` module at INFO level. Log: query type, params (truncated), result count, elapsed_ms.

    - Test at least some analogues have narrative (non-empty string) — validates Phase 4 Gemini cache data
    - Test NoDataError raised when structured_query returns empty (use a Cypher query that matches nothing)
    - Tests use live Neo4j Docker service (no mocks — locked decision)

       - Freshness threshold: 10 days (weekly)
       - Raises NoDataError if no rows for any requested series

    4. `get_gold_price(lookback_days: int = 7, now_override=None, engine=None) -> list[GoldPriceRow]`

       - Freshness threshold: 10 days (weekly)
       - Raises NoDataError if no rows
