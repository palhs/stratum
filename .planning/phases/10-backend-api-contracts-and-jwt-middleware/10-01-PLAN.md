3. **Create `reasoning/app/auth.py`:** Implement the `require_auth` FastAPI dependency:
   - Import `os`, `jwt` (from PyJWT), FastAPI's `Depends`, `HTTPException`, `status`, and `HTTPAuthorizationCredentials`, `HTTPBearer` from `fastapi.security`.
   - Instantiate `_bearer = HTTPBearer(auto_error=False)` â€” CRITICAL: `auto_error=False` prevents FastAPI's default 403 on missing header, allowing us to return 401 instead.
   - `async def require_auth(cred: HTTPAuthorizationCredentials | None = Depends(_bearer)) -> dict:` â€” returns decoded JWT payload.
   - If `cred is None`: raise `HTTPException(status_code=401, detail="Authorization header required", headers={"WWW-Authenticate": "Bearer"})`.
   - Read `secret = os.environ["SUPABASE_JWT_SECRET"]` (fail loudly if missing).
   - Call `jwt.decode(cred.credentials, secret, algorithms=["HS256"], audience="authenticated")`.
   - Catch `jwt.ExpiredSignatureError` -> 403 "Token expired".
   - Catch `jwt.InvalidAudienceError` -> 403 "Invalid token audience".
   - Catch `jwt.PyJWTError` (base class) -> 401 "Invalid token" with `WWW-Authenticate: Bearer` header.
   - Return the decoded payload dict on success.

4. **Create `reasoning/app/schemas.py`:** Define all Pydantic v2 response schemas used by both the OHLCV and report history endpoints (shared file avoids circular imports):
   ```python
   from pydantic import BaseModel
   from typing import Optional

   class OHLCVPoint(BaseModel):
       time: int              # Unix timestamp (seconds since epoch)
       open: Optional[float] = None
       high: Optional[float] = None
       low: Optional[float] = None
       close: float
       volume: Optional[int] = None
       ma50: Optional[float] = None
       ma200: Optional[float] = None

   class OHLCVResponse(BaseModel):
       symbol: str
       data: list[OHLCVPoint]

   class ReportHistoryItem(BaseModel):
       report_id: int
       generated_at: str      # ISO 8601
       tier: str              # "Favorable" | "Neutral" | "Cautious" | "Avoid"
       verdict: str           # one-line narrative

   class ReportHistoryResponse(BaseModel):
       symbol: str
       page: int
       per_page: int
       total: int
       items: list[ReportHistoryItem]
   ```
