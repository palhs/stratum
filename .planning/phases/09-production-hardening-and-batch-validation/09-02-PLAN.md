    **2. Determine Your Billing Path section:**
    Steps to check whether the GEMINI_API_KEY is linked to a Google Cloud project:
    - Go to AI Studio -> Settings -> Billing
    - If linked to Cloud project: follow Cloud Billing path (Section 3a)
    - If AI Studio only: follow AI Studio-only path (Section 3b)
    - Document this as a prerequisite decision

    **3a. Cloud Billing Budget Setup (recommended path):**
    Step-by-step instructions:
    1. Navigate to Google Cloud Console -> Billing -> Budgets & alerts
    2. Click "Create Budget"
    3. Scope: select the Cloud project linked to the Gemini API key
    4. Budget amount: $20/month (conservative starting point for single-user system)
       - Note: adjust based on actual usage after first batch run
    5. Alert threshold rules — add three rules:
       - 50% of budget (actual spend) — early warning
       - 80% of budget (actual spend) — action needed
       - 100% of budget (actual spend) — budget reached
    6. Notification: enable "Email alerts to billing admins and users"
    7. Optionally: add Cloud Monitoring notification channel for custom email
    8. Click "Finish" to create the budget

    **3b. AI Studio-Only Path (limited):**
    - Go to AI Studio -> Settings -> Billing -> Spend cap
    - Set monthly spend cap (e.g., $20/month)
    - Document limitation: spend cap is a hard stop with ~10 min billing lag, not a notification system
    - No tiered alerts available — document this explicitly as a known limitation

    **4. AI Studio Spend Cap (both paths):**
    - Set as safety backstop regardless of which billing path is used
    - Navigate to AI Studio -> Spend page
    - Set monthly spend cap value
    - Note: marked "experimental" — may change behavior

    **5. Testing Alerts section:**
    - Cloud Billing: use "Test notification" button in budget UI to verify email delivery
    - AI Studio: no test mechanism — verify by checking spend page after a small batch run
    - Document expected notification format (email subject line, sender)

    **6. Configuration Record section:**
    Template for recording actual values:
    ```
    Billing path:         [Cloud Project / AI Studio Only]
    Monthly budget:       $[amount]
    Threshold tiers:      50% ($[x]), 80% ($[y]), 100% ($[z])
    Notification email:   [email]
    AI Studio spend cap:  $[amount]
    Date configured:      [date]
    ```

    **7. Maintenance section:**
    - Review budget monthly during initial batch runs
    - Adjust budget amount based on actual usage patterns
    - Check AI Studio spend cap is still active (experimental feature)
