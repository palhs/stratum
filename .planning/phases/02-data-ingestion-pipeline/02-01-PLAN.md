`sidecar/app/routers/vnstock.py`:
- Pydantic models: IngestRequest (start_date, end_date, symbols optional, resolution optional default "weekly"), IngestResponse (status, rows_ingested, data_as_of, error_message optional, anomaly_detected bool)
- POST /ingest/vnstock/ohlcv — calls vnstock_service.fetch_and_upsert_ohlcv(), returns IngestResponse
- POST /ingest/vnstock/fundamentals — calls vnstock_service.fetch_and_upsert_fundamentals(), returns IngestResponse
- Error handling: catch vnstock API errors → 502, empty results → 204, unexpected → 500

`sidecar/app/services/vnstock_service.py`:
- `get_vn30_symbols()`: Use `Listing(source="vci").symbols_by_group(group="VN30")` — NEVER hard-code VN30 symbols
- `fetch_and_upsert_ohlcv(symbols, start_date, end_date, resolution, db_session)`:
  1. If symbols is None, call get_vn30_symbols()
  2. For each symbol: `Vnstock().stock(symbol=sym, source='VCI').quote.history(start=start, end=end, interval='1W')` for weekly
  3. Add symbol, resolution columns to DataFrame
  4. Map vnstock column names to our schema (time→data_as_of, open/high/low/close/volume)
  5. Add ingested_at = pd.Timestamp.now(tz="UTC")
  6. Upsert via INSERT ON CONFLICT DO UPDATE on (symbol, resolution, data_as_of)
  7. Return row count
- `fetch_and_upsert_fundamentals(symbols, db_session)`:
  1. If symbols is None, call get_vn30_symbols()
  2. For each symbol: `Vnstock().stock(symbol=sym, source='VCI').finance.ratio(period='year', lang='en', dropna=True)`
  3. Extract PE, PB, EPS, market cap, ROE, ROA, revenue growth, net margin
  4. Set data_as_of to the period end date from the financial data, ingested_at = NOW()
  5. Upsert to stock_fundamentals
  6. Return row count
- Historical backfill: 5 years on first run — the start_date parameter controls this (caller passes 5-year-ago date)
