**Step 2: Create `scripts/seed-neo4j-regimes.py`**

Python script that:
1. Reads `neo4j/seed/regime_data.json`
2. Connects to Neo4j via `neo4j` Python driver (bolt://neo4j:7687)
3. Uses UNWIND+MERGE pattern (from RESEARCH.md Pattern 1) to bulk-create Regime nodes idempotently
4. Reports count of nodes created/updated

Key implementation details:
- Use `session.execute_write()` with a transaction function
- MERGE on `{id: row.id}`, then `ON CREATE SET r += row, ON MATCH SET r += row`
- Read NEO4J_PASSWORD from environment variable
- Read NEO4J_URI from environment variable (default: bolt://neo4j:7687)
- Add `python-dotenv` for .env loading when run locally
- Print summary: "Seeded {N} regime nodes" with list of IDs
- Exit with code 0 on success, 1 on failure

Script header:
```python
#!/usr/bin/env python3
"""
seed-neo4j-regimes.py — Seed Neo4j with historical macro regime nodes.
Phase 4 | Plan 01 | Requirement: DATA-01

Usage:
    python scripts/seed-neo4j-regimes.py

Environment:
    NEO4J_URI      — Neo4j bolt URI (default: bolt://neo4j:7687)
    NEO4J_PASSWORD — Neo4j password (required)

Idempotent: MERGE ensures re-runs update existing nodes without duplicates.
"""
```
