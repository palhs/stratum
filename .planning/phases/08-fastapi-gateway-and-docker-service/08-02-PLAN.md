       **Background task function:**
       ```python
       async def _run_pipeline(job_id: int, ticker: str, asset_type: str, app_state):
           db_engine = app_state.db_engine
           try:
               _update_job_status(db_engine, job_id, "running")
               queue = app_state.job_queues.get(job_id)
               vi_id, en_id = await generate_report(
                   ticker=ticker,
                   asset_type=asset_type,
                   db_engine=db_engine,
                   neo4j_driver=app_state.neo4j_driver,
                   qdrant_client=app_state.qdrant_client,
                   db_uri=app_state.db_uri,
               )
               _update_job_status(db_engine, job_id, "completed", report_id=vi_id)
               if queue:
                   await queue.put(None)  # Sentinel for SSE completion
           except Exception as e:
               _update_job_status(db_engine, job_id, "failed", error=str(e))
               queue = app_state.job_queues.get(job_id)
               if queue:
                   await queue.put(None)  # Sentinel for SSE even on failure
       ```
       NOTE: report_id is set to vi_id (Vietnamese report — primary language per locked decision). The en_id is also stored in the reports table but the job FK points to the Vietnamese report as the primary.
