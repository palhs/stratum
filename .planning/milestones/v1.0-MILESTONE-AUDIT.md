### Phase 2: Data Ingestion Pipeline

- **TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID not passed to n8n container** — add to docker-compose.yml n8n environment block. Without this, all Telegram alert nodes fail silently.
- **WGC monthly retry wastes ~21 min/month** — monthly-wgc.json retry counter does not short-circuit on 501. Add IF node to skip retries on known 501 status.
- **FRED_API_KEY manual setup** — operators must set in .env.local; 9 tests skip and gold/FRED endpoints return 503 without it. Documented and handled gracefully (by design).
- **01-01-SUMMARY.md missing `requirements-completed` frontmatter** — no impact on verification but breaks 3-source cross-reference automation.

**Total: 5 items across 2 phases**
